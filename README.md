# AIbeaty

Monorepo for an AI operating system for beauty salons.

This repository is being prepared before the landing design is applied. It will hold both the public marketing site and the internal product system.

## Working Positioning

- Product: AI operating system for beauty salons
- Market: single-location salons, chains, studio owners, salon operators
- Entry offer: founder-led AI transformation audit plus first live workflow rollout
- Commercial promise: recover missed demand, automate follow-up, improve rebooking, give owners operational visibility

## Repository Layout

```text
AIbeaty/
├── apps/
│   ├── landing/     # public sales site
│   └── platform/    # internal operator and salon system
├── brand/           # naming, messaging, visual direction notes
├── docs/            # product, GTM, and architecture briefs
├── package.json
└── README.md
```
